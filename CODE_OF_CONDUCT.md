# 贡献者公约

本项目采 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。

## 我们的承诺

为营造一个包容环境, 我们作为贡献者与维护者承诺: 以尊重与同理思考走在项目里。

## 应该做

- 朋友 / 包容 / 考虑不同背景
- 建设性批评, 描述问题 而非人
- 以项目与社区最佳利益为重
- 不同意时 对事不对人

## 不应该做

- 人身攻击 / 侮辱 / 歧视
- 骚扰 (公开或私下)
- 公开他人隐私 (邮 / 住 / 身份)
- 任何不专业 / 不适宜于工作场合的行为

## 举报

被犯 / 看到犯规者, 发信 (占位): conduct@example.com。所有举报都会被迫切与公正调查。举报者隐私被保护。

## 后果

护者可依事重采:

1. **警告** — 私信提醒, 记在创。
2. **临时禁言** — 禁 1-30 日参与。
3. **永久禁** — 严重或多次犯规者。

被护者决定踩跰与本公约不一致, 可向项目 Steering Committee 上诉 (仓主)。
